L'application est un compteur de point de vie généralement utilisé pour les joueurs de jeu de cartes tel que "Yu-Gi-Oh!" "Magic" "Force of will"
et bien d'autres.
Le joueur met le téléphone au milieu de la table et les 2 joueurs peuvent interagir avec l'application.

Point principale :
- Compteur de point de vie
- Historique des coups joué
- Dé
- Minuteur

Feature :
- Jouer un coup (cliquer sur les points de vie que l'on souhaite modifier -> remplir le champ -> cliquer sur "+" ou "-" selon le choix -> cliquer sur "fin" pour fermer le pop-up)
- Possibilité d'annuler un coup joué (historique -> cliquer sur le(s) coup(s) que l'on souhaite annuler -> cliquer sur l'icône retour)
- Remettre les points de vie avec les valeurs d'origine (Cliquer sur les 3 traits en haut à droite -> cliquer sur "reset life point")
- Point de vie de départ configurable (Cliquer sur les 3 traits en haut à droite -> cliquer sur "setting" -> modifier les valeurs puis cliquer sur "finish")
	=> pour avoir la nouvelle valeur il faut remettre à zero le chrono (voir au-dessus)
- Lancer le minuteur (cliquer dessus)
- Mettre en pause le minuteur (cliquer dessus)
- Le minuteur sonne quand il arrive à zéro et il continue dans les négatifs, pour arrêter la sonnerie cliquée sur ok sur le pop-up qui s'ouvre.
- Remettre le minuteur aux valeurs d'origine (Cliquer sur les 3 traits en haut à droite -> cliquer sur "reset timer")
- Minuteur configurable (Cliquer sur les 3 traits en haut à droite -> cliquer sur "setting" -> modifier les valeurs puis cliquer sur "finish")
	=> pour avoir la nouvelle valeur il faut remettre à zéro les points de vie (voir au-dessus)
- Lancer le dé (cliquer dessus, l'image change en fonction du résultat)
- Bandeau publicitaire (généré par une api de google)
